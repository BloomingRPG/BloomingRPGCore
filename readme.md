# BloomingRPGCore
### OverView
このプロジェクトは制作中のMinecraftJEサーバー「BloomingRPG」の  
中心となるプラグインです。  

###Auto Build
Jenkinsを使用した自動ビルドでpush後約一分でデバックサーバーに反映されます。
